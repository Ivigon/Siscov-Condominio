# Siscov-Condominio
Siscov - Gestão de Portaria Sistema web autônomo para cadastro de visitantes e moradores.  Tecnologias: HTML5, CSS3 e JavaScript (ES6+).  Diferenciais: Armazenamento via LocalStorage (dados ficam salvos no PC), interface responsiva e Modo Noturno para conforto visual. Funciona 100% offline.
